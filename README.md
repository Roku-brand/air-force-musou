# 空軍無双

ブラウザだけで遊べる、依存なしの3D風戦闘機アクションゲームです。  
敵戦闘機だけを相手に、4つの広域空域を突破していきます。

## 特徴

- インストール不要で `index.html` を開くだけで起動
- HUD、レーダー、ロックオン表示つきの空戦シミュレーション風UI
- 4ステージ構成で、沿岸・洋上・山岳・市街地の敵航空戦力を殲滅するミッション
- ミサイル残弾の回復や被弾制限がある、アーケード寄りの戦闘バランス

## 起動方法

1. このフォルダを開く
2. `index.html` をブラウザで開く
3. 可能なら Chrome / Edge 系で全画面表示にすると遊びやすいです

## 操作方法

| キー | 内容 |
| --- | --- |
| `W / S` | 機首の上下 |
| `A / D` | 機体の左右傾き |
| `Q / E` | 旋回 |
| `Shift / Ctrl` | 加速 / 減速 |
| `↑ / ↓` | 上下移動 |
| `← / →` | 左右移動 |
| `Space` | ミサイル発射 |
| `R` | ステージ再出撃 |
| `Esc` | メニューへ戻る |

### スマホ操作

| UI | 内容 |
| --- | --- |
| 左スティック | 機首の上下 / 旋回 |
| 右スティック | 上下移動 / 左右移動 |
| `ROLL L / ROLL R` | 機体の左右傾き |
| `THR+ / THR-` | 加速 / 減速 |
| `FIRE` | ミサイル発射 |

## ゲームルール

- 耐久値は `3`
- `3` 発被弾すると撃墜
- 高度 `0m` で海面衝突（墜落）してゲームオーバー
- ミサイル残弾は `20`
- ミサイルは `3` 秒ごとに `1` 発ずつ回復
- ステージごとに敵配置と任務目標が変化

## ファイル構成

- `index.html`: 画面レイアウトとUI定義
- `styles.css`: HUD、オーバーレイ、全体スタイル
- `src/config.js`: ステージ設定、バランス調整値
- `src/entities.js`: 敵・味方・弾体などのゲーム内オブジェクト
- `src/math.js`: 3D計算用ユーティリティ
- `src/render.js`: 描画処理
- `src/game.js`: ゲーム進行、入力、UI更新
